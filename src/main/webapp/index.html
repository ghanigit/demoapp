🚀 Maven Demo Web App

Java 11 · Servlets · WAR · JUnit 5

👋 Hello Servlet

Calls GET /hello?name=... and returns a greeting.

🧮 Calculator Servlet

Calls POST /calculate, uses MathUtils internally.

📦 Maven Commands to Practice

CommandWhat it does
mvn testCompiles & runs JUnit 5 tests in src/test/
mvn packageRuns tests, then builds target/demo-webapp.war
mvn installSame as package + copies WAR to your local ~/.m2 repository
mvn cleanDeletes the target/ directory
mvn clean packageFresh build from scratch
